# UrbanFit - MERN Stack

Full MERN stack gym website with:

- React frontend in `client`
- Node.js + Express + MongoDB backend in `server`

## Setup

1. Install dependencies:
   - `cd client && npm install`
   - `cd ../server && npm install`
2. Create env files:
   - Copy `server/.env.example` to `server/.env` (set `JWT_SECRET`; install [MongoDB](https://www.mongodb.com/try/download/community) locally or use Atlas and set `MONGO_URI`)
   - Copy `client/.env.example` to `client/.env`
3. Run backend:
   - `cd server && npm run dev`
4. Run frontend:
   - `cd client && npm start`

Frontend: `http://localhost:3000`
Backend: `http://localhost:5000`
